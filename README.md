# Arnav Enterprises - Medicine Distributor Website

A complete, working website for Arnav Enterprises, a medicine distributor based in Pepsicola, Nepal with multiple pages and full navigation.

## Pages

1. **index.html** - Home page with hero section, features, and call-to-action
2. **about.html** - About Us page with company story, mission, values, team, and coverage map
3. **products.html** - Products page showcasing medicine categories (Tablets, Capsules, Syrups, Injections, Ointments, Pediatric)
4. **contact.html** - Contact page with contact information, map, and contact form
5. **get-supply.html** - Inquiry form page for requesting medicine supply quotes

## Features

- ✅ Fully responsive design (mobile, tablet, desktop)
- ✅ Dark mode support
- ✅ Working navigation between all pages
- ✅ Mobile menu functionality
- ✅ Contact forms with validation
- ✅ WhatsApp floating button on all pages
- ✅ Consistent branding and styling across all pages
- ✅ Modern UI with Tailwind CSS
- ✅ Material Symbols icons

## How to Use

1. Open `index.html` in your web browser to start
2. Navigate between pages using the header navigation menu
3. Use the mobile menu button (☰) on mobile devices
4. Fill out forms on Contact and Get Supply pages
5. All pages are linked and working together
